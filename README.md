# Design-and-Verification-of-IC-555-Timer-Using-RTL-UVM-Testbench-and-Physical-Design-Flow
This project implements a complete VLSI flow for the IC 555 Timer, from RTL modeling in Verilog to UVM-based functional verification. It covers the full backend Physical Design cycle, including floorplanning, CTS, and routing, ensuring timing and power closure for a layout-ready silicon implementation.
